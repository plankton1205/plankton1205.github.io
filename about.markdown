Welcome，dear reader! I'm [@plankton1205](https://substack.com/@plankton1205/) on Substack.

Right now I dominantly operate my newsletter on [Substack](https://substack.com/) (come subscribe and interact with me!). But the idea is: this page will always be the ultimate backup space for all the blogs I’ve ever posted. You can subscribe to my current platform, or come here, find the RSS link at the bottom of the home page, or send me an email to be manually added to my Mailchimp list. That is what this address for, a weirdo and nerdy apocalypse shelter, prepared more as a looks-cool side project than out of any real worry. This website is hosted on GitHub and build with Jekyll, so it will stand as long as GitHub stands. Jekyll is a nice static website generator, check it out if you’re curious about building a website and not too afraid of getting your hands dirty with code. ;)

For all the layout I'm just using the base Jekyll theme. You can find out more info about customizing your Jekyll theme, as well as basic Jekyll usage documentation at [jekyllrb.com](https://jekyllrb.com/)
